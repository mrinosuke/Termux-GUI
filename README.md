## Installation [Termux]:

```bash
apt-get update -y

apt-get upgrade -y

apt install git -y

git clone https://github.com/mrinosuke/Termux-GUI

ls

cd Termux-GUI

chmod +x ternux_gui.sh

./termux_gui.sh
```

### Note command: 
1) 'vnc'     - for start vnc server
2) 'vncstop' - for stop vnc server
